# 🎨 Pygame Paint (TSIS 2)

Графический редактор, созданный на Python с использованием библиотеки Pygame. Проект является расширенной версией классического приложения Paint и реализует рисование от руки, создание геометрических фигур, заливку, ввод текста и сохранение готового рисунка.

## 🌟 Возможности (Features)

- **Инструменты рисования:** Карандаш (свободное рисование), Прямая линия, Ластик.
- **Геометрические фигуры:** Прямоугольник, Квадрат, Круг, Прямоугольный треугольник, Равносторонний треугольник, Ромб.
  - *Фигуры имеют прозрачный предпросмотр (live preview) во время растягивания мышью.*
- **Заливка (Flood-Fill):** Закрашивание замкнутых областей выбранным цветом (реализовано через собственный алгоритм на основе стека).
- **Текстовый инструмент:** Ввод текста прямо на холсте в реальном времени.
- **Настройка кисти:** 3 уровня толщины (маленькая, средняя, большая). Работает для карандаша, ластика и всех фигур.
- **Сохранение:** Экспорт рисунка в формате `.png` с автоматическим добавлением времени (timestamp) в имя файла.
